## ¿Qué fuente utilizas para programar?

Si eres parte de la mayoría que utiliza Visual Studio como su entorno de desarrollo integrado (IDE), probablemente utilices Consolas, siempre y cuando no la hayas cambiado. Y esa es una buena fuente, ¿por qué querrías cambiarla? Bueno, todo se reduce a preferencias personales.

## Fira Code

Fira Code se basa en Fira Mono de Mozilla. Tiene una altura x regular, muchas ligaduras de codificación y destaca en legibilidad. Eso es lo que lo convierte en una mejor línea de base para la comparación. Así es como se ve en acción.

## Commit Mono

Commit Mono es descrito por sus autores como _una fuente de codificación anónima y neutral centrada en mejorar la experiencia de lectura._ Es una fuente variable, altamente personalizable a través de sus numerosas funciones de Open Type.

Incluso cuenta con "kerning inteligente". El kerning es crucial para la legibilidad, pero hasta ahora ha sido imposible combinarlo con fuentes de espaciado fijo. El "kerning inteligente" desplaza las letras a una mejor posición mientras conserva el espaciado fijo. Esto hace que Commit Mono sea una fuente monoespaciada altamente legible.

## Cascadia

Cascadia es la nueva fuente monoespaciada predeterminada de Microsoft. Viene incluida con Windows Terminal y es la opción predeterminada en Visual Studio. Entonces, si estás utilizando la última versión de VS y no has cambiado tu fuente predeterminada, ¿quizás estás usando esta pero no eres consciente?

La versión en cursiva también tiene un estilo alternativo, cursivo, que se ve genial para comentarios en el código. Así es como se ve en acción.

## Victor Mono

Sus autores lo describen como una fuente monoespaciada de código abierto con cursiva opcional semi-conectada y ligaduras de símbolos de programación. Es esbelto, nítido y estrecho, con una altura x grande y una puntuación clara, lo que lo hace legible e ideal para el código. Viene en siete pesos y estilos Romano, Itálico y Oblicuo.

A diferencia de Cascadia, el estilo cursivo no es opcional con Victor Mono. Es así como está diseñado el estilo Itálico.

## SF Mono

Esta es la fuente monoespaciada predeterminada de macOS pero "ligaturizada". Esto significa que se tomó la SF Mono de Apple y se le agregaron las ligaduras de Fira Code con Ligaturizer. Es una herramienta para hacer exactamente eso. Entonces, si ya has elegido tu fuente favorita y no tiene ligaduras, aún puedes agregarlas con esta herramienta.

## Recursive

Recursive es la última fuente en mi lista para examinar, pero puede ser la más impresionante. Es una fuente con cinco propiedades variables: peso, inclinación, mono, casual y cursiva. Esto significa que es altamente personalizable.

Las propiedades variables mono, casual y cursiva son las más interesantes. Mono cambia la fuente de completamente monoespaciada a proporcional (normal) (ejemplo inferior en la Figura 24). La propiedad Casual hace que la fuente parezca casi como el estilo de escritura (tercer ejemplo en la Figura 24). Es lo opuesto a la configuración Lineal. La última es cursiva, que cambia algunos de los caracteres a la variante cursiva.
